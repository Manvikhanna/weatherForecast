## Weather Forecast App

### Features

- City search with debounce
- Current weather & 5-day forecast
- Celsius / Fahrenheit toggle
- Recent searches
- Dark mode
- Geolocation weather

### Tech Stack

- Next.js
- React
- OpenWeatherMap API

### Setup

1. npm install
2. Add API key to .env.local
3. npm run dev
